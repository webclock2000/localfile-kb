# FileKB

File-driven personal knowledge base — 100% local, zero cloud.

Monitor local directories, auto-extract structured knowledge, build a queryable
knowledge graph, and ask natural language questions with full source provenance.

> **Status**: Early development. Design phase complete. Implementation in progress.

## Quick Start (planned)

```bash
pip install filekb
filekb index --watch
filekb ask "What decisions did I make about the API design?"
```

See [docs/](docs/) for design documents.
